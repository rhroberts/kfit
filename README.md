# kfit - A simple spectral fitting tool in Python

## Installation

### The easy way

`sudo snap install kfit`

### From source

#### Dependencies

From pip/conda...

- lmfit
- matplotlib
- pandas
- numpy

From apt or similar...

- python3-gi
- python3-gi-cairo
- gir1.2-gtk-3.0
